今天，小破网站又开张啦
更新于 2020-12-05 日常 热度 1 喜欢 0

我平常是一个喜欢观察喜欢思考的人，也因此想法/思考/感悟会比较多，很多时候都想把随时而来的idea记录下来，但又苦于没有养成写字的习惯，很是无奈。仔细回想为什么没有养成码字的习惯呢，我觉得主要有以下两个原因：
  ①找不到喜欢的码字工具/平台。之前一直都有在知乎上写东西，但是越来越不喜欢知乎的审核机制和推荐机制，体验太差，所以索性还是放弃了。
  ②很多时候尽管码字会让我思如泉涌，但也会使得我的大脑持续兴奋很久，每当我码完字以后都会难于入睡（由于大脑持续兴奋导致失眠）

直到去年（2019年底）这个时候，刚搬出去实习，无意中接触到博客，觉得非常有意思非常新鲜，因为拥有一个博客就意味着能够随心所欲搭建属于自己的网站，也能随心所欲地记录自己的一切，想想都觉得十分美好。所以当时就在网上查了很多关于搭建博客的教程，最后用“wordpress + 国外服务器 + 国外域名”搭了我的第一个博客网站，开心之余我还给它起了个名字—小破网站。

记得那会每天晚上9，10点下班，从公司回到出租房差不多10点半到11点的样子，看会书，洗漱下到12点，然后开始哼哧哼哧地修整着自己的小破网站，简简单单并快快乐乐，时常会感觉到时间飞逝，没弄一会就到凌晨2、3点，通常也是毫无困意。不过由于第二天还得上班的缘故，所以我也会强迫着自己躺下休息…

睡前我经常会在博客上简单记录一下当天的心情，现在还依稀记得那会每天记录最多的感受就是挫败感+焦虑，由于第一次出去实习+基础较差的缘故，每天都会面临各种无法解决的bug，再加上leader高强度的push工作，导致我时常心力交瘁（通俗点就是在领导的要求下每天的工作要有进展，但是经常会遇到几个程序上的问题无法解决导致进度条一直卡着不动），这种无法调和的矛盾导致我每天都会很焦虑，很多时候都是怀揣着忐忑的心情下班（没办法我太菜了）…

也因此，每天最快乐的时光，就是洗漱完后坐在电脑面前的那一小段时间，简单纯粹而且成就感满满，尤其是看着我的小破网站一点一点按照自己的想法布置起来的时候，这种感觉就像全身心培养一个自己的孩子一样，倾注心血、见证一点一滴的成长。

有时候我们说什么样的生活是简单幸福的，这段经历给的我个人的体会是：一份普通的工作 + 一份热爱的小事。

老实说我非常享受一个人独处的时光，因为能有大把的时光去追逐一切我好奇感兴趣的事物，能够有无尽的思考与感悟的同时还能够更加接近生活本质。

（此处省略1万字生活感悟…）

感悟之余，在实习结束后，由于即将面临秋招，我没有多少时间去管我的小破网站，也因此它最终的结局是被关停了。

其实当时遗弃的原因有好几个，主要是因为域名和服务器都布置在国外，无论是修改还是访问都需要翻墙，十分不方便，再加上wordpress把博客功能封装的过于细致，导致细节修改十分麻烦，所以最终也就不了了之。

这次我借鉴了之前搭博客的经验，重新花了大概一周的时间搭建了这个新的小破网站（我称之为小破网站2.0哈哈），希望自己不要懒惰，忙碌之余也要坚持，希望自己能够真正将热爱转化成习惯。

最后我想说：再次遇见你真好，这一次，一起加油吧！

end.
